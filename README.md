# Phaser 彈幕射擊遊戲

一個使用 Phaser 3 + TypeScript + Vite 開發的進階彈幕射擊遊戲（STG）。

## 技術特色

- ⚡ **Vite** - 快速開發與建置
- 🎮 **Phaser 3** - 遊戲引擎
- 📘 **TypeScript** - 型別安全
- 💥 **複雜彈幕系統** - 多種彈幕模式
- 🎨 **粒子特效** - 視覺效果
- 🎵 **音效系統** - 背景音樂與音效
- 📊 **性能優化** - 物件池、紋理圖集

## 開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 控制方式

- **方向鍵 / WASD** - 移動玩家
- **空格鍵 / Z** - 射擊
- **Shift** - 慢速移動（精確控制）
- **ESC** - 暫停/選單

## 遊戲特色

- 多種彈幕模式（螺旋、波形、圓形、追蹤）
- 連擊系統與分數計算
- 生命值系統
- 敵人 AI 與行為模式
- 攝像機震動效果
- 粒子特效系統
